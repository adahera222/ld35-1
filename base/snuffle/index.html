program:
Rules:
  • f - forward
  • b - back
  • <space> - pause
  • r - right turn
  • l - left turn
  • p - port step (left)
  • s - starboard step (right)
Push Enter to reload.

Future rules:
  • <^v> - absolute directions
  • ! - next instruction is immediate
  • (...) - group instructions
  • [...] - pick random instruction
  • <number> - execute next instruction n times
